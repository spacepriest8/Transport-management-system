# Renaissance Transport Management System (RTMS)

A modern, high-performance web application designed to streamline journey planning, vehicle tracking, and personnel manifest management. 

## 🚀 Overview
RTMS is built to manage the logistics of moving personnel safely and efficiently. It connects vehicles, drivers, and employees through a centralized Supabase database, offering real-time insights into transport operations.

## ✨ Key Features
- **Live Dashboard:** Real-time KPIs showing today's journeys, active transits, and vehicle availability.
- **Journey Planning:** Link drivers, vehicles, and multiple passengers to specific routes with estimated arrival/departure tracking.
- **Personnel Management:** Track employee clearance levels, departments, and company affiliations.
- **Vehicle & Driver Database:** Monitor fleet status (Active/Maintenance) and driver availability.
- **Automated Manifests:** Generate and download PDF manifests for every scheduled journey.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid), JavaScript (ES6+)
- **Backend/Database:** [Supabase](https://supabase.com/) (PostgreSQL & Real-time)
- **Icons:** [Tabler Icons](https://tabler-icons.io/)
- **PDF Generation:** jsPDF & jsPDF-AutoTable
